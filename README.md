# FamilyMealPlanner

מתכנן ארוחות משפחתי – אתר סטטי (client-side, ללא שרת) שעוזר לבחור מה יש בבית, להגדיר העדפות משפחתיות, ולקבל רעיונות לארוחות, תפריט שבועי ורשימת קניות.

🔗 האתר החי: https://odelya100.github.io/FamilyMealPlanner/

הפרויקט בנוי מקובץ HTML אחד (`index.html`) הטוען React ו-Babel מ-CDN בזמן ריצה, ולכן אינו דורש בנייה (build) או שרת – ניתן להריץ אותו ישירות על ידי פתיחת index.html, או כאתר סטטי ב-GitHub Pages.
